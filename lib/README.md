# Student Task Organizer - منظم المهام الدراسية

تطبيق Flutter يساعد الطلاب على تنظيم المهام الدراسية، مع إحصائيات، بحث، فلترة، وإدارة المواد.

## الميزات
- عرض المواد بالألوان والأيقونات
- إضافة/حذف/تعديل المهام
- البحث والفلترة حسب الحالة
- الترتيب حسب التاريخ أو الأولوية
- إحصائيات مفصلة (نسبة الإنجاز، توزيع المهام)

## التقنيات
- Flutter
- Dart
- intl
- flutter_localizations

## التشغيل
```bash
flutter pub get
flutter run
